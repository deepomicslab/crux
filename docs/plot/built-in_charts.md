# Step Line 

The `StepLine` will take the parsed data of the XYPlot.

```bvt
XYPlot {
    StepLine {};
}
```
### Step Line Options

> stroke: `string`
> fill: `string`

> pathOptions: `any`
A dictionary of the other styles you would like to apply to the path.
> closedEnd: `boolean`

> closedStart: `boolean`

Arc {
                            x1 = d.start; x2 = d.end
                            r1 = innerRadius
                            r2 = 100%
                            fill = d.color
                            
                        }
